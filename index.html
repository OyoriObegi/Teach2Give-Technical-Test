✦ FAQs

What is Frontend Mentor, and how will it help me?

+

Frontend Mentor offers realistic coding challenges to help developers improve their front-end coding skills with projects in HTML, CSS, and JavaScript. It's suitable for all levels and ideal for portfolio building.

Is Frontend Mentor free?

+

Yes, Frontend Mentor offers both free and premium coding challenges.

Can I use Frontend Mentor projects in my portfolio?

+

Yes, you can use projects completed on Frontend Mentor in your portfolio to showcase your skills.

How can I get help if I'm stuck on a challenge?

+

You can ask questions in the Frontend Mentor Slack community or browse solutions submitted by others for inspiration.
